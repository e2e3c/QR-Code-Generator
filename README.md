# QR Code Generator

A simple Python application that generates QR codes from URLs or text using the `qrcode` library.

## Features

- Generate QR codes from text or URLs
- Save the QR code as a PNG image
- Easy to use
- Lightweight and beginner-friendly

## Requirements

- Python 3.x
- qrcode
- Pillow

## Installation

```bash
pip install qrcode[pil]
```

## Usage

Run the program:

```bash
python main.py
```

Enter a URL or any text when prompted:

```text
Enter URL: https://github.com/e2e3c
```

The generated QR code will be saved as:

```text
qrcode.png
```
